# Kevin Bakan - Portfolio

Detta är min portfoliosida som jag har gjort i skolan. Jag är nybörjare inom webbutveckling och håller på att lära mig HTML, CSS och JavaScript.

## Länk till sidan
Har inte lagt upp den än men ska göra det snart.

## Vad är det här för projekt?
Det är en enkel hemsida om mig själv där man kan se vem jag är, vad jag kan och kontakta mig. Jag har också lagt till en knapp där man kan ladda ner mitt CV.

## Vad har jag använt?
- HTML - för att bygga själva sidan
- CSS - för att göra den snygg och ge den färger
- JavaScript - har inte använt så mycket än men filen finns där
- Font Awesome - det är ikonerna som github och linkedin symbolerna
- Google Fonts - för att få snyggare text på sidan
- FormSubmit - för kontaktformuläret, vet inte helt hur det fungerar men det ska skicka mail

## Hur har jag byggt den?
Jag använde VS Code för att skriva koden. Jag började med HTML strukturen och lade sedan till CSS för att styla den. Jag försökte göra den responsiv vilket betyder att den ska fungera på mobil också, inte säker på om det funkar helt perfekt.

Jag använde något som heter Flexbox i CSS för att placera saker bredvid varandra, det var lite förvirrande i början men tror jag förstår det nu ungefär.

## Filer i projektet
- index.html - hela hemsidan
- style.css - all styling
- script.js - javascript, mest tom just nu
- headshot.webp.jpg - min profilbild
- Kevin Bakan CV NIBE.pdf - mitt CV som man kan ladda ner

- Bilden ville inte ladda för att filnamnet var fel
- Kontaktformuläret fungerar inte när man öppnar filen lokalt, måste läggas upp på internet
- Har haft problem med att taggar försvunnit när jag kopierat kod
